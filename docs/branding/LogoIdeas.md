# Logo Ideas

## Direction 1 — Wordmark

A clean wordmark using:

```text
B3YOND
CO.
```

## Direction 2 — Symbol + Wordmark

A geometric symbol representing:

- Connection
- Motion
- Progress
- Going beyond limits

## Direction 3 — The 3 as Identity

Use the "3" as a visual anchor in the logo.

Possible meanings:

- Build
- Connect
- Go Beyond

## Notes

Do not finalize logo before trademark and brand research.
