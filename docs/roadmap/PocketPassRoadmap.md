# PocketPass Roadmap

## Product Vision

PocketPass helps people reconnect with the meaningful people they meet in real life.

## Problem Statement

People naturally meet interesting people every day, but many of those connections disappear because exchanging contact information is awkward, forgotten, or never happens.

PocketPass removes that friction by helping users reconnect after genuine real-world encounters.

## MVP Features

- User accounts
- Public profile cards
- Bluetooth-based encounter detection
- Encounter history
- Connection requests
- Event Mode
- Privacy settings

## Future Features

- Event organizer tools
- Professional networking mode
- Dating mode
- Community discovery
- Digital business cards
- AI-assisted networking
- Team and enterprise features

## Product North Star

**Meaningful Connections Created**

Downloads are not enough. PocketPass succeeds when it helps people create genuine, valuable connections.
